```
Realice la descarga e instalación del framework Phoenix y de PostgreSQL.

Ya quedo todo corriendo al crear el servidor.
```

```
## Modulo 2
Ya se realizó toda la lectura de este módulo y se realizaron los cambios en base a los ejemplos de la guía de
cada tema, en la cual se nos hablo de los temas: Directory structure, Request life-cycle, Plug, Routing,
Controllers y Views and templates.
```
```
## Modulo 3
Ya se realizó todo el estudio del módulo 3, que es un poco extenso, en el cual se habla más delos aspectos para
realizar un carrito de compras con las validaciones necesarias y del cual se tocan muchos temas importantes como lo
son los: config repo, the schema, changesets and validations, persistence, context, In-context Relationships,
Cross-context dependencies, entre otras.

La aplicación quedo funcional con todos los pasos de la guía.
```
```
## Modulo 4
Ya se realizó todo el estudio del módulo 4, en el cual nos habla de casi todo lo que uno necesita para construir y
mantener una aplicación web pero introduce muchos conceptos nuevos que toca repasar en otras ocasiones para no
olvidarlo. Se tocan muchos temas importantes como lo son los: Understanding test modules, The ConnCase, Running
tests using tags, Testing posts, The DataCase, HTML controller tests, JSON controller tests, entre otras.

La aplicación sigue funcional aunque al tratarse de una guía que no tiene como requisito las anteriores muchos de
los test presentan errores.
```
